## Hero background

`roof-stock.jpeg` is a **stock shingle close-up** used only as a muted, navy-tinted
backdrop behind the hero text — it is decorative (empty `alt`) and is **not**
presented as a Rodriguez Roofing job. All photos in the "Our Work" gallery are the
business's own real job photos.

## Honesty note

The gallery labels each photo by **stage** (On the job / During / Finished). These
are **different projects**, not a single project's before-and-after — so no two
unrelated photos are presented as the same roof's before & after.
